# Grovr

A Git Worktree Manager for macOS.

## Features

- **Worktree Management** - Create, delete, and switch between worktrees
- **IDE Integration** - Open in VS Code, Cursor, Zed, or your preferred editor
- **GitHub/Jira** - View PR status and issue badges inline
- **Keyboard Shortcuts** - `⌘,` settings, `⌘V` quick create from clipboard

## Download

Get the latest release from [Releases](https://github.com/j1king/grovr-desktop/releases).

## Development

```bash
pnpm install
pnpm tauri dev
```

## Tech Stack

Tauri v2 · React 19 · TypeScript · TailwindCSS · Rust
